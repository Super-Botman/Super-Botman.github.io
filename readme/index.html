README

A quick reference for every shortcut and command in this file browser.

Table of Contents

  • 1. Shortcuts
  • 2. Commands

Shortcuts

KeyAction
shift+lSwitch focus to the file viewer
shift+hSwitch focus to the file browser
jScroll down a file, or select the next file in the browser
kScroll up a file, or select the previous file in the browser
lScroll right
hScroll left
enterOpen the selected file — or run a command, if you're typing one (see Commands)
escFocus the command prompt
shift+tOpen the selected file or folder in a new tab
shift+qClose the current tab
tabMove to the next link on the page (standard browser behavior)

Commands

Press esc to focus the prompt, then type a command below.

CommandDescription
:helpShow this page
:qLeave the site
:set mouse=true|falseShow (true) or hide (false) the mouse cursor
